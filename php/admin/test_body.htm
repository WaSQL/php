Data-navigate allows you to use your keyboard to navigate through your web app. Just add the following attributes to any HTML element:

data-navigate="1"
data-navigate-group="yournavgroup"
data-navigate-focus="Alt+m" - the keyboard combination to use to access this element. Alt and Ctrl are supported
data-navigate-up="yourCustomJavascriptFunction();" - run a custom javascript function when this element has focus and the up key is pressed. "up, down, left, right, enter, and tab" are all supported
<span class="navigate-text">Element Text</span> - by placing this class around the text, the access key will be underlined with red for Ctrl or blue for Alt access keys

Test this by hitting Ctrl+d using your keyboard to set the focus to the first element below then use the arrow keys to navigate through the menu.

There is also a javascript function called navigateSetFocus where you can set the focus as follows buildOnLoad("navigateSetFocus('Ctrl+D');")

There is also a javascript function called navigateHideAndSetFocus where you can hide and id and set the focus as follows navigateHideAndSetFocus('run_detail_menu','Alt+T')
