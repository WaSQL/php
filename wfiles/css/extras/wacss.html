WaCSS Navbar - Responsive Navigation
  • Clean Bulma-like styling - Professional appearance with smooth hover effects
  • 3-level dropdown support - Nested ul elements up to 3 levels deep
  • Desktop hover navigation - Dropdowns appear on hover for desktop users
  • Mobile-responsive design - Hamburger menu for mobile and tablet devices
  • Touch-friendly - Click-to-open dropdowns on mobile devices
  • Semantic HTML structure - Uses proper ul/li elements throughout
  • Brand/logo area - Dedicated space for site branding
  • Right-aligned items - Support for login/signup buttons
  • Dropdown dividers - Visual separation in dropdown menus
  • Smooth animations - CSS transitions for better user experience

With Color Variant

<nav class="wacss_navbar is-primary">

With Size Variant

<nav class="wacss_navbar is-small">

Usage

To use this navbar, simply apply the wacss_navbar class to your ul element and structure your navigation using nested ul/li elements:

<ul class="wacss_navbar"> <li class="navbar-brand"> <a href="#">Brand</a> </li> <button class="navbar-burger"> <span></span> <span></span> <span></span> </button> <ul class="navbar-menu"> <li><a href="#">Home</a></li> <li class="has-dropdown"> <a href="#">Services</a> <ul> <li><a href="#">Web Design</a></li> <li><a href="#">Development</a></li> </ul> </li> </ul> </ul>

Key classes:

  • wacss_navbar - Main navbar container
  • navbar-brand - Brand/logo area
  • navbar-burger - Mobile hamburger menu button
  • navbar-menu - Main navigation container
  • has-dropdown - Apply to li elements that contain nested ul
  • navbar-end - Right-align navigation items

Responsive Behavior

Desktop (1024px+): Horizontal layout with hover dropdowns

Mobile/Tablet (<1024px): Collapsible menu with click-to-open dropdowns

Touch devices: Hover effects are disabled, dropdowns open on click
